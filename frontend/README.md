# PID — Onde Investir?

Frontend Next.js + React + TypeScript para a demo “Onde Investir?” da PID.

## Rodar localmente

Instale dependências com Bun:

```bash
bun install
```

Suba o servidor de desenvolvimento:

```bash
bun dev
```

Valide build e tipos:

```bash
bun run typecheck
bun run build
```
